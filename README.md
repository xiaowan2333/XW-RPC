### XW-RPC
手写RPC框架练习
![image](https://user-images.githubusercontent.com/48382305/180402031-440412a6-66b7-4878-bded-9a94d07bce28.png)

***
###已实现功能
1. 支持多种服务远程调用
2. 内置BIO和NIO网络通信模式
3. 支持多种序列化方式
4. 支持多种服务注册中心，目前只实现了nacos
5. 支持服务端调用负载均衡
***

3. 定义注解进行服务暴露和服务代理
4. 实现服务列表的缓存
